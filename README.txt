README — Customs Photo Site

Файлы:
- index.html — страница, показывающая фото
- customs-decl.jpg — ваше фото (название содержит "customs")
- qr.html — страница для генерации QR-кода на ПРЯМОЙ URL картинки
- README — эта инструкция

Публикация на GitHub Pages:
1) Создайте публичный репозиторий, например: customs-photo
2) Загрузите ВСЕ файлы из этой папки в корень репозитория и сделайте Commit.
3) Settings → Pages:
   • Source: Deploy from a branch
   • Branch: main / (root)
   • Save
4) Итоговые адреса:
   Сайт:   https://USERNAME.github.io/customs-photo/
   Фото:   https://USERNAME.github.io/customs-photo/customs-decl.jpg
   QR-паге: https://USERNAME.github.io/customs-photo/qr.html?u=https%3A%2F%2FUSERNAME.github.io%2Fcustoms-photo%2Fcustoms-decl.jpg
   (замените USERNAME и имя репозитория при необходимости)

Получение QR:
- Откройте ссылку на QR-страницу выше — она сразу покажет QR и предложит скачать PNG.
